# Backend & Fullstack Projects

A collection of my backend and fullstack projects. Each project lives in its own repository.

## Projects

| Project | Description | Stack |
|--------|-------------|-------|
| [Spiral Sound](https://github.com/ThuyKat/Spiral-Sound) | Vinyl record store — browse catalogue, manage cart, and check out via Stripe. Includes user authentication, session management, and a webhook handler for order fulfilment. Migrated from plain HTML/CSS/vanilla JS (ES modules) to React. | React, Stripe, HTML, CSS, JavaScript |
| [VendorPal BE](https://github.com/ThuyKat/VendorPal-BE) | REST API backend for a vendor point-of-sale and shop management app. Features JWT auth, email verification, password reset, RBAC, multi-tenancy, product/category/order management, and Gmail SMTP integration via OAuth 2.0. | Java, Spring Boot, Spring Security, MySQL, JPA/Hibernate, Docker |
| [VendorPal FE](https://github.com/ThuyKat/VendorPal-FE) | React frontend for VendorPal. Includes a POS system, admin dashboard with analytics and reporting, product/category/order management, and JWT-based auth with protected routes and auto-logout. | React, Vite, React Router, Axios, CSS Modules |
